# Christina Wright Art Portfolio

A responsive professional art portfolio website for traditional paintings, hand-drawn art, digital art, and web/game design projects.

## Files

- `index.html` — main website structure
- `style.css` — full responsive styling
- `script.js` — mobile menu, gallery filters, and image lightbox
- `images/` — artwork samples

## How to preview in VS Code

1. Open this folder in Visual Studio Code.
2. Install the Live Server extension if you do not already have it.
3. Right-click `index.html`.
4. Choose **Open with Live Server**.

## How to publish on GitHub Pages

1. Upload all files to a GitHub repository.
2. Go to repository **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.
